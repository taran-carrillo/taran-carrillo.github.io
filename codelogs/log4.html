How does CSS accept color as a value? Provide examples of colors in these styles:

Named: CSS has a data type that represents pre-set values as the color names, and the only thing you have to enter is the name of the color instead of a number value.

Hexadecimal: The hexadecimal is a series of numbers and letters that describe a syntax of an RGB color using its primary color components. (Red, Green, Blue)

RGB: or sRGB (standard red, green, blue), uses the same primary colors and white point combined in different ways to display a typical viewing experience.

HSL: or Hue, Saturation, and lightness, defines the components of an RGB color through defining each level seperately.
