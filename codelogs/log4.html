CSS Color

What do these CSS properties do?

  • Color: Sets a new property different from the default on whatever tag it is attached to.
  • Background-color: Adds a color to the padding around the text or image the tag is attached to, or the background of the entire page if no tag is attached.
  • Border-color: adds a color to the border as it is defined by another tag.

How does CSS accept color as a value? Provide examples of colors in these styles:

Named: CSS has a data type that represents pre-set values as the color names, and the only thing you have to enter is the name of the color instead of a number value.

Hexadecimal: The hexadecimal is a series of numbers and letters that describe a syntax of an RGB color using its primary color components(Red, Green, Blue). It is defined by choosing values between 00 as the lowest and FF as the highest value for each primary color, the hexadecimal value is specified with #RRGGBB. So a hexadecimal that looks like this: #FF0000, would be a solid red color because the red section is set to the max and the others are set at 0.

RGB: or sRGB (standard red, green, blue), uses the same primary colors and white point combined in different ways to display a typical viewing experience. RGB can be set as values between 0 and 255 or as a percentage up to 100%.

HSL: or Hue, Saturation, and lightness, defines the components of an RGB color through defining each level seperately. Hue is based on a number 0-360 based on the colors' location on the color wheel. Saturation is a percentage 0-100, 0 being a shade of gray and 100 being the full color. Lightness is also on a 0-100 percent scale, 0 being dark and 100 being light.
