How does a CSS media query work, and why is it useful?

answer

What "media features" are supported by media queries and what do they do?

Print. The print media query feature allows the designer to target what happens when the page is being printed out. It is useful when you want something on your webpage, but you don't want it printed out. It can be useful for images that are unnessecary to print but add to the overall webpage

Max-width. This feature allows the webpage to change an element when the screen size is larger than the provided unit. This is useful for when you know you are working with both small devices, such as a phone, and larger ones, such as a computer or monitor.
