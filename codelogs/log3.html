CSS Media Queries

How does a CSS media query work, and why is it useful?

They target specific functions of browsers and devices to apply styles that affect those browsers based on each attributed unit of measure or function. It is useful when creating websites that are accessed on different devices with different shapes and sizes. Responsive design is made possible by CSS media queries, they are the tools that developers need to create a responsive piece of media that can be shared through various platforms.

What "media features" are supported by media queries and what do they do?

Print.
The print media query feature allows the designer to target what happens when the page is being printed out. It is useful when you want something on your webpage, but you don't want it printed out. It can be useful for images that are unnessecary to print but add to the overall webpage

Max-width.
This feature allows the webpage to change an element when the screen size is larger than the provided unit. This is useful for when you know you are working with both small devices, such as a phone, and larger ones, such as a computer or monitor.

Orientation.
Depending on how your screen is oriented, landscape or portrait, the website can display aspects and ratios in a better suited manner. It aligns the layout to your screen, vertical or horizontal.

Pointer.
This feature allows the web developer to easily identify how each form of selecting, a mouse, a touchscreen, or something else, interacts with the webpage. The pointer query is followed by a description of how accurate the pointer is; such as, coarse, or a finger on a touchscreen, or fine, a cursor.
