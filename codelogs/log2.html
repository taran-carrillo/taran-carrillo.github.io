Code Log 2:CSS Measuring Units

What is the difference between "Absolute" and "Relative" CSS length units?

  • The difference is that relative units rely on other units to give them a value whereas absolute units are generally considered to always be the same.
  • The benefit to using relative units instead of a set unit comes when you are attempting to scale something in comparison to another element. Say you want some shape to be exactly scaled to the other one, you can use a relative unit instead of trying to match with an absolute unit.
